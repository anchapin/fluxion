       - Create HTML with two subplots (2 rows, 1 column). Upper chart: temperature series (multiple zones). Lower chart: HVAC (positive/negative) and maybe solar gains.
       - Include JavaScript control elements: play/pause button (`<button>Play</button>`), speed input (`<input type="number" value="1">` hours/second), and a scrubber (`<input type="range" min="0" max="8759">`).

         * `let currentHour = 0; let speed = 1; let interval;`
         * `function renderFrame(hour) { update charts to show data up to hour; }`

         * Pause clears interval.
         * Scrubber oninput sets currentHour and calls renderFrame.
       - Initial render shows full data; during animation, we can set `xaxis.range` to [0, currentHour] or update data arrays to first `currentHour` points. Plotly's `restyle` can update data; using `relayout` to change xaxis range is efficient.
