2. Add stack effect ACH calculation function:
```rust
/// Calculates air change rate (ACH) due to stack effect through door opening.
///
/// Stack effect is thermal buoyancy-driven airflow proportional to √(ΔT/h).
/// Warmer air rises, creating pressure difference that drives airflow.
///
/// # Arguments
/// * `temp_a` - Temperature in zone A (°C)
/// * `temp_b` - Temperature in zone B (°C)
/// * `door_height` - Door opening height (meters)
/// * `door_area` - Door opening area (m²)
///
/// # Returns
/// Air change rate ACH (1/hour, dimensionless)
///
/// # Formula
/// Volumetric flow rate: Q_vent = C·A_door·√(ΔT / h_door)
/// ACH = Q_vent / V_zone = Q_vent / (A_door · h_door)
///
/// Where:
/// - C = STACK_COEFFICIENT = 0.025 (empirical, may need calibration)
/// - ΔT = |T_A - T_B| (absolute temperature difference, K)
/// - h_door = door height (m)
/// - A_door = door area (m²)
/// - V_zone = A_door · h_door (approximate zone volume)
///
/// # Temperature Dependence
/// ACH ∝ √(ΔT), not linear. Small ΔT → small ACH, large ΔT → large ACH.
/// Example: ΔT = 10°C → ACH ∝ 3.16, ΔT = 20°C → ACH ∝ 4.47 (42% higher)
///
/// # Example
/// ```rust
/// use fluxion::sim::interzone::calculate_stack_effect_ach;
///
/// // Back-zone (20°C) to sunspace (0°C), winter condition
/// let ach = calculate_stack_effect_ach(20.0, 0.0, 2.0, 1.5);
/// // Q_vent = 0.025 * 1.5 * √(20/2.0) = 0.119 m³/hr
/// // ACH = 0.119 / (1.5 * 2.0) = 0.04 /hr (very low, closed door)
/// ```
pub fn calculate_stack_effect_ach(
    temp_a: f64,
    temp_b: f64,
    door_height: f64,
    door_area: f64,
) -> f64 {
    // Temperature difference (absolute value for magnitude)
    let delta_t = (temp_a - temp_b).abs();

    // Stack effect volumetric flow rate: Q_vent = C·A_door·√(ΔT/h_door)
    let q_vent = STACK_COEFFICIENT * door_area * (delta_t / door_height).sqrt();

    // ACH = Q_vent / V_zone
    // V_zone = A_door · h_door (approximate zone volume near door)
    let zone_volume = door_area * door_height;

    q_vent / zone_volume
}
```

3. Add ventilation heat transfer function:

/// Calculates ventilation heat transfer using air enthalpy method.
///
/// # Arguments
/// * `ach` - Air change rate (1/hour)
/// * `temp_source` - Source zone temperature (°C)
/// * `temp_target` - Target zone temperature (°C)
/// * `volume_target` - Target zone volume (m³)
///
/// # Returns
/// Ventilation heat transfer Q_vent (Watts). Positive if heat flows source→target.
///
/// # Formula
/// Q_vent = ρ·Cp·ACH·V_target·(T_source - T_target)
///
/// Where:
/// - ρ = AIR_DENSITY = 1.2 kg/m³
/// - Cp = AIR_SPECIFIC_HEAT = 1000 J/kg·K
/// - Units: (kg/m³)·(J/kg·K)·(1/hr)·(m³)·K = J/hr
/// - Convert to Watts: /3600
///
/// # Common Pitfall
/// Omitting air density and specific heat gives 1200x error:
/// Q_wrong = ACH·V·ΔT (m³/hr) instead of Q_correct = ρ·Cp·ACH·V·ΔT (W)
///
/// # Example
/// ```rust
/// use fluxion::sim::interzone::calculate_ventilation_heat_transfer;
///
/// // Sunspace (30°C) to back-zone (20°C), ACH = 1.0 /hr, V = 100 m³
/// let q = calculate_ventilation_heat_transfer(1.0, 30.0, 20.0, 100.0);
/// // Q = 1.2 * 1000 * 1.0 * 100 * 10 / 3600 = 333 W
/// ```
pub fn calculate_ventilation_heat_transfer(
    ach: f64,
    temp_source: f64,
    temp_target: f64,
    volume_target: f64,
) -> f64 {
    let delta_t = temp_source - temp_target;

    // Air enthalpy method: Q_vent = ρ·Cp·ACH·V·(T_source - T_target)
    // Result in J/hr, convert to Watts: /3600
    (AIR_DENSITY * AIR_SPECIFIC_HEAT * ach * volume_target * delta_t) / 3600.0
}
```
