Create unit tests for temperature-dependent air exchange rate (ACH) using stack effect:

1. Test stack effect ACH formula:
   - Temperature A = 20°C, Temperature B = 0°C (ΔT = 20°C, winter condition)
   - Door height = 2.0 m (typical), Door area = 1.5 m² (typical)
   - Stack effect: Q_vent = 0.025 * A_door * √(ΔT/h_door)
     - Q_vent = 0.025 * 1.5 * √(20/2.0) = 0.025 * 1.5 * 3.16 = 0.119 m³/hr
   - Zone volume = A_door * h_door = 1.5 * 2.0 = 3.0 m³ (approximate)
   - ACH = Q_vent / V_zone = 0.119 / 3.0 = 0.04 /hr (very low, closed door)
   - Verify ACH calculation matches expected formula

2. Test air enthalpy heat transfer method:
   - ACH = 0.04 /hr, Volume = 3.0 m³, ΔT = 20°C
   - Air density ρ = 1.2 kg/m³, Specific heat Cp = 1000 J/kgK
   - Q_vent = ρ·Cp·ACH·V·ΔT = 1.2 * 1000 * 0.04 * 3.0 * 20 = 2880 J/hr = 0.8 W
   - Verify ventilation heat transfer includes air density and specific heat
   - Common pitfall: omitting ρ·Cp gives 1200x error

3. Test temperature dependence:
   - ΔT = 0°C → ACH = 0 /hr (no buoyancy, equal temps)
   - ΔT = 10°C → ACH proportional to √10 ≈ 3.16
   - ΔT = 20°C → ACH proportional to √20 ≈ 4.47 (42% higher than ΔT=10)
   - Verify ACH increases with temperature difference (√(ΔT) relationship)
