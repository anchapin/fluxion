# Plan 05-04: Systematic Issues Analysis and Reporting

## Objective

Analyze validation failures across all ASHRAE 140 cases to identify common failure patterns, categorize systematic issues, and track progress toward 100% validation compliance. Produce consolidated documentation:
- `docs/KNOWN_ISSUES.md` - Catalog of all known systematic issues with severity and resolution status
- `docs/QUALITY_METRICS.md` - Metrics dashboard tracking pass rate, MAE, and deviation trends over time

This addresses requirement REPORT-04.

## Background

As validation progresses, failures fall into distinct categories (e.g., solar gain issues, inter-zone radiation, ground coupling). Documenting these systematically helps:
- Prioritize fix efforts (high-severity, high-frequency issues first)
- Avoid redundant work (recognize when different cases show same root cause)
- Communicate status to stakeholders
- Track progress toward validation goals

## Locked Decisions

1. **Issue Taxonomy:**
   - `BASE-01` through `BASE-04`: Foundation (conductance, HVAC, weather)
   - `SOLAR-01` through `SOLAR-04`: Solar radiation and shading
   - `FREE-01` through `FREE-03`: Free-floating temperature validation
   - `TEMP-01` through `TEMP-03`: Temperature swing and thermal lag
   - `MULTI-01`: Multi-zone inter-zone transfer
   - `GROUND-01`: Ground boundary conditions
   - `REPORT-01` through `REPORT-04`: Reporting and diagnostics (meta)

2. **Severity Levels:**
   - **Critical:** Causes >50% failure rate across multiple cases
   - **High:** Causes 10-50% failure rate or single high-impact case failure
   - **Medium:** Causes 1-10% failures or affects non-primary metrics
   - **Low:** Cosmetic or edge case issues

3. **Documentation Format:**
   - Markdown tables for easy editing
   - Each issue has: ID, description, affected metrics, severity, GitHub issue link, resolution status, phase addressed
   - Include historical metrics (before/after comparisons)

4. **Metrics Computation:**
   - Pass rate: `passed / total` (cases)
   - MAE: Mean Absolute Error across all numeric metrics
   - Max Deviation: Worst single metric deviation
   - Track by phase and cumulatively
